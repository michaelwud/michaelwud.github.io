学校c语言大赛赛后补题

Posted on 2019-04-05 |
Words count in article: 1.1k | Reading time ≈ 5

Topic

毒瘤代码
时间限制: 1 Sec 内存限制: 128 MB
题目描述

Dye认为虽然代码能够AC，但是还不够完美，所以在Dye提交代码之前，Dye打算尽可能的缩写他的变量名。Dye的代码里有n个变量名，变量名两两互不相同，且只包含小写字母’a’-‘z’。

他希望用非空前缀替换每个变量名，使得这些新变量名仍然是互不相同的（但是，某些变量的新名称可以与另一个或同一个变量的旧名称重合）。他想找到一种新名字总长度最小的方法。

*字符串a 是字符串b的前缀，表示你可以从b的末尾删除若干个(或0个)字符并获得a 。

请找出新变量名的最小可能总长度。

输入

第一行包含一个整数N(1≤N≤10**5)，表示变量的个数

接着N行表示N个变量名，变量名只包含小写字母’a’-‘z’。

变量名的总长度不超过10**5，且互不相同。

输出

输出一行，包含一个整数——新变量名的最小可能总长度

样例输入 样例输出 提示

样例中我们用”aac”作为最后一个变量名，将”a”作为第一个变量名，剩下的变量不需要改变，得到最小总长度为11

Contents

如题意所描述

Solution

  • 先建trie图，发现问题变成将带有终结标记的点往上挪使得所有结点挪动的高度差值之和最大。
  • 对问题进行子问题分解，那么对于一棵树，我们知道，若以根节点root的儿子为根的子树都已经最优，那么我们此时只需要在以根节点root的子树上找深度最大的点去放在根结点就可以了（要先保证根节点root无终结标记）。这样子每个子树都是最优，进而推到整个树才会是最优的。
  • 在放置的时候将深度最大点和root的深度差累加进答案。
  • 因为需要知道子树下深度最大点以及更新标记位置，那么需要
    • dfs序：知道以root为根的子树范围
    • 线段树：维护dfs序下区间最大值（即深度最大），同时修改终结标记

这个题比赛的时候一听完题意就有想法了，然后因为太久没敲题了，略有生疏，就懒得打了，赛后就补了一下，就当作练练手。
